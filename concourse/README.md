# Concourse CI

* [Concourse Helm chart documentation] (https://github.com/kubernetes/charts/tree/master/stable/concourse)
* [Concourse tutorial](https://concoursetutorial.com)

## Install Concourse Helm Chart

```sh
make install
```

## Upgrade via Helm

```sh
make upgrade
```

## Uninstall

```sh
make uninstall
```
